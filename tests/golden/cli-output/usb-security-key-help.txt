USB security-key CTAP/WebAuthn proxy status, sessions, and cancellation

Usage: d2b usb security-key <COMMAND>

Commands:
  status    Show configured security keys, per-VM virtual-device health, and active lease
  sessions  List active and recent security-key request sessions
  cancel    Cancel an in-progress or queued security-key request
  test      Smoke-check virtual HID presence in a VM and host broker reachability
  help      Print this message or the help of the given subcommand(s)

Options:
  -h, --help  Print help
